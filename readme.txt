== Description ==

The Plugin helps you to add the SVG animation using the Lottie Files. This animations can be easily integrated with any themes and builders. The file format is tiny, high quality and can be manipulated at runtime. It also works in any devices and screensizes. 

== Major features in R Animated Icon Plugin include: ==

1. SVG Animation

2. Lottie Animation

3. Can be used as standard shortcode into any theme or page builders.

4. File uploaded through the Media can be added to the Shortcode by copying the URL from the media file

5. Free Lottie animation files can be found <a href="https://lottiefiles.com/featured">Here</a>

== Shortcode ==

[r_animated_icon_default_shortcode r_animated_icon_file="" r_animated_icon_extra_class=""]

== Installation ==
 
This section describes how to install the plugin and get it working.

1. Go to Wordpress Dashboard > Plugins > click on Installed plugins > look for add new at the top most of the screen and click here > Add Plugins > upload plugin > choose file -> click here and upload R Animated Icon plugin.

2. Activate Plugin through 'Plugins' menu in WordPress.

== Manual installation via FTP ==
1. Download the plugin file to your computer and unzip it

2. Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation's wp-content/plugins/ directory.

3. Activate the plugin from the Plugins menu within the WordPress admin
